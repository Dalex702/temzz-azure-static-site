# Static Website Deployment on Azure

## Project Overview
This project demonstrates the deployment of a static website using **Azure Static Web Apps** and **GitHub Actions** for CI/CD automation.

## Architecture
- **GitHub Repository**: Hosts the website source code and workflow files.
- **Azure Static Web App**: Hosts the deployed website.
- **GitHub Actions**: Automates build and deployment.

## Deployment Steps
1. Created GitHub repository `temzz-azure-static-site`.
2. Added website files (HTML, CSS, JS).
3. Created Azure Static Web App.
4. Linked the GitHub repository to Azure.
5. Configured workflow for automatic deployment.
6. Verified deployment via the live URL.
